## ✨ المميزات

### 🎨 التصميم
- **Glassmorphism** — تصميم زجاجي شفاف على جميع العناصر
- **Dark / Light Mode** — تبديل فوري مع حفظ الإعداد
- **Syntax Highlighting** — تلوين كود JavaScript تلقائياً بدون مكتبات خارجية
- **Ambient Orbs** — خلفية متحركة ثلاثية الأبعاد
- **Grain Texture** — طبقة texture احترافية فوق الخلفية

### ⚡ الأداء
- **Lazy Loading** — كل فصل يُحمَّل عند فتحه فقط
- **يتذكر آخر فصل** — يعود إليه تلقائياً عند كل زيارة
- **يعمل أوفلاين** — بعد أول تحميل

### 📱 تجربة المستخدم
- **نسخ بضغطة واحدة** — مع toast notification فوري
- **Progress Bar** — يتتبع الأسئلة المفتوحة في كل فصل
- **Swipe بين الفصول** — على الموبايل يميناً ويساراً
- **Scroll to Top** — زرار يظهر عند التمرير للأسفل
- **بحث سريع** — في قائمة الفصول الجانبية
- **Expand / Collapse All** — فتح وإغلاق جميع الأسئلة دفعة واحدة
